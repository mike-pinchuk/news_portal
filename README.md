## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
This is an application news portal, where user able to watch posted news by another users and see general information (e.g. weather, currency etc...)

## Running the app
#### run docker-compose
```bash
$ docker-compose up -d mydb
```

#### run localy
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
After this, the server should be available at http://localhost:3000 from the host machine.
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
